四个核心页面

🏠 首页 /
  • AI 搜索框（跨会议提问）
  • 按日期分组的会议列表
  • 新录音按钮
  • 筛选：文件夹、日期范围
📝 文档页 /meeting/[id]
  • 当前 page.tsx 的核心内容
  • NoteEditor + EnhancedNotes
  • 底部浮动工具栏（录音/转写/Chat）
  • 浮动面板（转写、Chat）
💬 AI 对话 /chat
  • 全屏 AI 对话界面
  • 跨会议知识库问答
  • 来源引用 + 跳转到会议
  • 对话历史列表
⚙️ 设置 /settings
  • ASR 配置（阿里云等）
  • LLM 配置（API Key、模型选择）
  • 模板管理
  • MCP 生态接入

路由结构

app/
├── layout.tsx ← 根 layout（html/body）
├── (main)/ ← 路由组：共享侧边栏
│ ├── layout.tsx ← Sidebar + {children}
│ ├── page.tsx ← 首页（会议列表）
│ ├── meeting/[id]/
│ │ └── page.tsx ← 文档编辑页
│ ├── chat/
│ │ └── page.tsx ← AI 对话页
│ └── settings/
│ └── page.tsx ← 设置页
└── api/ ← 现有 API 不变
