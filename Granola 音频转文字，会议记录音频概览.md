# Granola产品记录与思考

**音频转文字，会议记录/音频概览**

任何音视频，它都会后台监听转写文本。一键摘要总结（支持自定义Prompt） **可能是最好的实时语音转写 + 笔记工具。**



## 核心功能



- **无机器人静默录制**：这是 Granola 最大的特色之一。它不需要像其他工具那样派遣一个“虚拟机器人”加入会议频道，而是直接在后台录制并保存本地音频。这避免了会议中出现机器人的突兀感（但仍建议出于礼貌提前告知与会者）。
- **人机协作式笔记**：在录音的同时，你可以手动输入自己的笔记。会议结束后，Granola 会将你手动记下的要点与 AI 生成的完整转录文本进行对比和融合，从而梳理出结构最符合你思路的会议纪要。







使用场景：让AI陪着一起视频，语音记笔记，最后AI总结对话提炼。

1. 打开Youtube视频 
2. Granola新建一条笔记（自动开始监听声音） 
3. 听到有感触或重点，按Fn调用AI语音输入法记录。 
4. Granola AI总结是英文的，再用自定义Prompt重写笔记。

把 AI 当学习伙伴，它听、你想、它记、你问。最关键的是你保持了主动思考，AI 只是辅助记录和提炼。这比单纯看视频+做笔记效率高得多。



需要识别参会人或公司名称，为不同类型的会议创建提示词或模板



Granola 的新功能 Recipes，感觉就是提示词库？

1. 把专业提示词与真实工作场景结合， Chat 里用 “/” 调用。   
2. 上线高质量 Recipes（如 Lenny Rachitsky、Matt Mochary、Ridd 等），业界高手的提示词。 
3. 提示词场景： “Coach me” 让AI给工作反馈、“Prep me” 会前准备、“Write a brief”  生成会议文档等 
4. 可以创建、分享自己的 Recipes。
